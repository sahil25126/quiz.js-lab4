# JavaScript Quiz Game

This project is a simple JavaScript quiz game that runs entirely inside the **browser console**.  
The quiz uses arrays, loops, conditionals, functions, and input/output methods such as `prompt()` and `alert()`.

## Features
- Predefined questions stored in an array  
- Uses a `for` loop to show questions one by one  
- Accepts answers using `prompt()`  
- Input is normalized using `toLowerCase()` and `trim()`  
- Immediate feedback using `alert()`  
- Tracks total score  
- Displays final result  

## How to Run
1. Open Google Chrome (or any browser).  
2. Press **F12** → Open **Console**.  
3. Copy all the code from `quiz.js`.  
4. Paste it in the console → Press **Enter**.  
5. Answer the prompts and get your score!
